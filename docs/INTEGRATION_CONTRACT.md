# ATCB v0.3 Integration Contract

ATCB v0.3 is a standalone local continuity-governance service.

It is not THAL.
It is not Asteris.
It is not Nova.
It is not the mind.

ATCB is the continuity control plane around the mind.

## Allowed Calls

External systems may submit:

- claims
- memories
- clarifications
- review decisions
- boundary profiles

## ATCB May Return

- agent and thread state
- contradiction records
- confidence warnings
- recovery blocks
- audit trails
- review requirements

## Boundary

ATCB does not own the external system identity.
ATCB does not import private corpus by default.
ATCB does not claim to be the agent.
ATCB does not generate final conversational identity.
ATCB does not expand autonomy for any agent.

## Data Rule

Default operation uses generic configurable records only. Private THAL/Nova, Asteris/Olympus/Enki, or personal identity material must not be ingested unless a later explicit contract and review process allows it.
